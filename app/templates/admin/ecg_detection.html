About ECG Detection

This system uses an ensemble of deep learning models to detect cardiac arrhythmias:

  • CNN - Convolutional Neural Network
  • LSTM - Long Short-Term Memory
  • CNN-LSTM - Hybrid model
  • Random Forest - Traditional ML

Trained on MIT-BIH Arrhythmia Database (360 Hz sampling rate).
