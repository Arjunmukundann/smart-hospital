# 🏥 Smart Hospital Platform

A comprehensive hospital management system with AI-powered safety features, digital prescriptions, and intelligent analytics.

## ✨ Features

### Doctor Module
- 📋 Patient registration with medical survey
- 💊 Digital prescription creation
- 🔒 Real-time safety alerts (allergies, drug interactions)
- 📄 Medical report upload & AI analysis
- 🔍 OCR for handwritten prescriptions

### Patient Module
- 📱 Digital prescription view
- 🍽️ Meal safety analyzer
- ⏰ Medicine reminders
- 📝 Simplified medical summaries

### Admin Module
- 📊 Hospital analytics dashboard
- 📈 ML-based patient inflow predictions
- 💊 Medicine inventory management
- ⚠️ Safety alert logs & analysis
